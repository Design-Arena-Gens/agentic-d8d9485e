## Streamline – YouTube-inspired experience

Streamline is a polished video browsing and viewing experience built with Next.js 14, Tailwind CSS, and the App Router. It ships with curated video data, powerful search, category filters, and a fully featured watch page.

### Features

- Modern landing experience with hero spotlight, responsive grid, and category chips
- Search that updates URL parameters and filters the catalog in real time
- Dedicated watch page with embedded YouTube player, rich metadata, comments, and an “Up next” rail
- Responsive navigation crafted with Tailwind CSS and `lucide-react` icons

### Local development

```bash
npm install
npm run dev
# then open http://localhost:3000
```

### Production build

```bash
npm run build
npm start
```

Deploy the project on [Vercel](https://vercel.com) for a seamless production workflow.
